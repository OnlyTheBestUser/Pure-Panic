7. Core API Reference | PS4 Specific

APIs and types for PS4 platform, see fmod_ps4.h

  • FMOD_PS4_GetPadVolume Retrieves the output volume of the pad speaker as set by the user in the system software.
  • FMOD_PS4_SetComputeDevice Provide access to graphics resource to allow offloading to the GPU.
  • FMOD_PS4_ReleaseComputeDevice Release all GPU compute resources and no longer offload processing.
  • FMOD_PS4_SetThreadAffinity Control which core particular threads are created on.
  • FMOD_PS4_THREADAFFINITY Mapping of cores to threads.
  • FMOD_THREAD Cores available for mapping threads onto.
  • FMOD_PS4_PORT_TYPE Port types avaliable for routing audio.

FMOD_PS4_GetPadVolume

Retrieves the output volume of the pad speaker as set by the user in the system software.

C
C++
C#

FMOD_RESULT FMOD_PS4_GetPadVolume(
  FMOD_SYSTEM *system,
  SceUserServiceUserId userID,
  float *volume
);

Not supported for C#.

Not supported for JavaScript.

system
Handle. (System)
userID
Logged in user identifier. (SceUserServiceUserId)
volume Out
Output parameter to receive the volume.

There must be at least one channel group attached to the users pad speaker for this function to work.

FMOD_PS4_PORT_TYPE

Port types avaliable for routing audio.

C
C++
C#

typedef enum FMOD_PS4_PORT_TYPE
{
  FMOD_PS4_PORT_TYPE_MUSIC,
  FMOD_PS4_PORT_TYPE_VOICE,
  FMOD_PS4_PORT_TYPE_PERSONAL,
  FMOD_PS4_PORT_TYPE_CONTROLLER,
  FMOD_PS4_PORT_TYPE_COPYRIGHT_MUSIC,
  FMOD_PS4_PORT_TYPE_SOCIAL,
  FMOD_PS4_PORT_TYPE_MAX
} FMOD_PS4_PORT_TYPE;

Not supported for C#.

Not supported for JavaScript.

FMOD_PS4_PORT_TYPE_MUSIC
Background music, pass FMOD_PORT_INDEX_NONE as port index.
FMOD_PS4_PORT_TYPE_VOICE
Voice chat, pass SceUserServiceUserId of desired user as port index.
FMOD_PS4_PORT_TYPE_PERSONAL
Personal audio device, pass SceUserServiceUserId of desired user as port index.
FMOD_PS4_PORT_TYPE_CONTROLLER
Controller speaker, pass SceUserServiceUserId of desired user as port index.
FMOD_PS4_PORT_TYPE_COPYRIGHT_MUSIC
Copyright background music, pass FMOD_PORT_INDEX_NONE as port index. You cannot have both copyright and non-copyright music open simultaneously.
FMOD_PS4_PORT_TYPE_SOCIAL
VR only - social screen when in separate mode otherwise not output, pass FMOD_PORT_INDEX_NONE as port index.
FMOD_PS4_PORT_TYPE_MAX
Maximum number of port types supported.

See Also: System::AttachChannelGroupToPort

C#

  FMOD_PS4_THREADAFFINITY *affinity
);

Not supported for C#.

Not supported for JavaScript.

Affinity for each thread. (FMOD_PS4_THREADAFFINITY)

Call before System::init or affinity values will not apply.
